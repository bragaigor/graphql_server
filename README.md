# Frist GraphQL Rust Server

### The following works with Rust nightly build

### Steps

Uninstall current Rust version
```
rustup self uninstall
```
Install nightly rust
```
curl https://sh.rustup.rs -sSf | sh
```

Nightly build with cargo
```
cargo +nightly build
```

Run the server
```
cargo +nightly run
```
